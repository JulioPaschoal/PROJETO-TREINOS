## 📦 API Treinos

API RESTful desenvolvida com Node.js, TypeScript e Fastify seguindo a arquitetura MVC, com foco em boas práticas, segurança, escalabilidade e padronização de código.

> 🚀 Tecnologias Utilizadas

- Node.js

- TypeScript

- fastify

- Arquitetura MVC

- Docker

- Eslint/Prettier – Padronização e formatação de código

> 📦 Clone o repositório:

```
git clone https://github.com/seu-usuario/sua-api.git
```

> 📂 Entre na pasta do projeto:

```
cd sua-api
```

> 🟢 Instale as dependências:

```
pnpm add
```

> ▶️ Executar o Projeto

```

pnpm dev
```

> 🖥️ Servidor rodando em:

- http://localhost:4000

> 📖 Documentação Swagger

Após iniciar o servidor, acesse a documentação interativa:

- http://localhost:4000/docs

> 🔐 Autenticação

- Biblioteca Better-Auth

> ✅ Validação de Dados

- Zod

> 🧹 Padronização de Código

- O projeto utiliza Eslint e Prettier

- Formatação automática

- Padronização de código

> 🐳 Docker

O projeto é containerizado com Docker, facilitando o ambiente de desenvolvimento e deploy.

> 🗄️ Banco de Dados

- Sequelize: Banco relacional PostgreSQL
